# stock-machine-v003

### Setup
- `sudo easy_install virtualenv`
- `virtualenv venv`
- `source venv/bin/activate`
- `pip install https://github.com/pallets/flask/tarball/master`
- `pip install -r requirements.txt`

- `npm install`
- `bower install`
- `gulp prod`

### Run
- `source venv/bin/activate`
- `python server.py`
